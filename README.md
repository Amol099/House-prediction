# House Price Prediction using Linear Regression

A Machine Learning web application built with Streamlit that predicts house prices based on property size using a Linear Regression model. The application provides real-time predictions along with an interactive visualization of the dataset and predicted value.

🚀 Overview

This project demonstrates a simple Machine Learning workflow using Linear Regression to estimate house prices based on house size.

The application is developed with Streamlit, allowing users to enter a house size, generate an estimated market price, and visualize the prediction on an interactive scatter plot.

It serves as an excellent beginner-friendly project for understanding regression models, model training, prediction, and deployment using Python.
